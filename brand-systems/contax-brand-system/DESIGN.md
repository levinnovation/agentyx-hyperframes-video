# Design System: Contax

Official design system specification for the **Contax** product brand.

## 1. Visual Atmosphere
*   **Atmosphere:** Deep violet/dark purple canvas, glowing magenta accents.
*   **Fonts:** Inherited from the core Agentyx Design System (Plus Jakarta Sans and Inter).

## 2. Usage Rules
Contax is a sibling product of Agentyx under Lev Innovation. It inherits the structural components (grids, cards, lists) from the parent design systems while using custom violet/magenta color tokens for its visual identity.

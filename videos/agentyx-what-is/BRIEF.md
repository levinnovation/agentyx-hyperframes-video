# Brief — Agentyx What Is

## Project

- **Name:** agentyx-what-is
- **Owner:** Agentyx / Lev Innovation
- **Date:** 2026-06-29
- **Source:** brand-system + pitch deck messaging
- **Target duration:** 25 seconds
- **Format:** landscape 1920×1080
- **Language:** Español
- **Distribution:** website, LinkedIn, sales, internal explainer

## Goal

Explicar qué es Agentyx en 25 segundos: ejecución operativa autónoma, no chatbot.

## Audience

Operadores B2B, directores de operaciones y equipos que ya conocen "IA" pero no distinguen asistencia vs ejecución.

## One Message

Agentyx ejecuta trabajo operativo real dentro de tus sistemas — no conversa contigo.

## CTA

Cierre de marca suave. Sin "agendar demo" fuerte.

## Must Include

- Logo Agentyx (apertura y cierre)
- Hub de integraciones (ERP, CRM, WhatsApp, Email)
- Pipeline Trigger → Context → Execute → Outcome
- KPIs operativos (24/7, minutos, -70% carga)
- Tema oscuro brand-system + mockups `.ax-light`

## Avoid

- Paleta clara del promo viejo
- Neon / clichés genéricos de IA
- Lucide icons
- CTA agresivo de demo
- Stock footage

## QA Notes

- Lint / validate / inspect / snapshot antes de render
- Preview: `npm run preview:what-is`

# Videos

Each video project should live in its own folder:

```text
videos/<video-name>/
├── index.html
├── DESIGN.md
├── SCRIPT.md
├── STORYBOARD.md
├── narration.txt
├── narration.wav
├── transcript.json
├── assets/
├── capture/
├── compositions/
├── snapshots/
└── renders/
```

The current reference project is:

- `agentyx-product-promo`

Use it as the canonical example for structure, naming, and quality checks.
